Visual Studio

Visual Studio is an Integrated Development Environment(IDE). It is developed by Microsoft to create Desktop applications, GUI (Graphical User Interface), console, web applications, mobile applications, cloud, and web services, ect. You can use this to write codes like HTML, JavaScript, Python, CSS, ect. Visual Studio doesn't need money for you to use it. Visual Studio is a free application.

Setting It Up

To get this awesome application to work, after downloading it, press on the file button up top and press "Open Folder...". Pick the folder you want to load in Visual Studio and press "Select Folder". If you already have your files downloaded into a folder, this makes it easier for you to open it. If you don't have your files ready, you can make new ones. To do that you make an empty folder in the folders place, and open it in Visual Studio. Then you click on the "new file" icon near the place where the word "explorer" is found. By clicking on the icon, you can name the files you want to make. Ex: index.html, main.js, style.css, ect. The next step is typing your code into the computer language you chose. For example, aftr creating a html file, you type html tags into enpty space on the right of the explorer side. The same goes with other languages like JavaScript and CSS.


Console Screen

Console Screen is mostly used as a debugging tool. Whenever you open console, it shows you if you have any mistakes in your code. It can also be used to show how the code functions. For example, if you choose JavaScript, and open the console page, you will see the funtions you used. Some function name are Concat function, Math max function, Square Root function, ect. To open console, you right click on the tab and press the word "inspect". Moving forward click on the console button on the top. If you follow these steps, you can see the console screen.
